# The-Day-To-Know

News App: Android

Basic news reel app using api to get a daily news via api call.  

Search functionality via search bar.  

## Features:
- Jetpack Compose
- Kotlin
- MVI
- Koin
- Ktor
- Coil
- Room
- Offline Capability
- Custom animations
- Modules  ( data -> domain <- presentation principle)
